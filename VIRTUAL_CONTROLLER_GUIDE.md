# 虚拟手柄控制界面使用指南

## 概述

虚拟手柄控制界面提供了一个图形化的方式来控制机械臂和AGV小车，无需物理手柄即可进行操作。

## 界面布局

```
┌─────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────┤

│ [Gripper Open] [Gripper Close]  [Pump On] [Pump Off]      │
```

## 功能说明

### 顶部按钮

#### 夹爪控制
- **Gripper Open** (绿色)：打开机械臂夹爪
- **Gripper Close** (绿色)：关闭机械臂夹爪

#### 泵控制
- **Pump On** (绿色)：打开吸泵
- **Pump Off** (绿色)：关闭吸泵

#### 机械臂锁定
- **Lock Arm** (红色)：锁定机械臂伺服电机
- **Release Arm** (红色)：释放机械臂伺服电机（可手动移动）

### 虚拟摇杆

#### 左侧 - 机械臂控制摇杆
控制MechArm 270机械臂的移动：

- **Arm X+** (右)：机械臂向右移动
- **Arm X-** (左)：机械臂向左移动
- **Arm Y+** (上)：机械臂向前移动
- **Arm Y-** (下)：机械臂向后移动

**操作方式**：
1. 用鼠标点击并拖动摇杆中心的蓝色圆球
2. 圆球会跟随鼠标移动
3. 松开鼠标，圆球自动回到中心
4. 实时显示当前位置：`Arm Control: X=0.0, Y=0.0`

#### 右侧 - AGV小车控制摇杆
控制AGV小车的移动：

- **Forward** (上)：小车前进
- **Backward** (下)：小车后退
- **Left** (左)：小车左转
- **Right** (右)：小车右转

**操作方式**：
1. 用鼠标点击并拖动摇杆中心的蓝色圆球
2. 圆球会跟随鼠标移动
3. 松开鼠标，圆球自动回到中心，小车停止
4. 实时显示当前位置：`AGV Control: X=0.0, Y=0.0`

### 底部按钮

#### Reset to Start Position (橙色)
- 重置机械臂到初始位置
- 重置两个摇杆到中心
- 停止所有运动

## 使用流程

### 基本操作流程

1. **进入虚拟手柄界面**
   - 从主菜单点击 "Virtual Controller Control"

2. **控制机械臂**
   - 点击 "Lock Arm" 锁定机械臂
   - 使用左侧摇杆控制机械臂移动
   - 使用 "Gripper Open/Close" 控制夹爪
   - 使用 "Pump On/Off" 控制吸泵

3. **控制AGV小车**
   - 使用右侧摇杆控制小车移动
   - 向上推：前进
   - 向下推：后退
   - 向左/右推：转向

4. **返回主菜单**
   - 点击顶部 "Intelligent Logistics System Virtual Controller Control" 按钮

### 抓取物体示例

1. 锁定机械臂：点击 "Lock Arm"
2. 移动到物体上方：使用左侧摇杆
3. 打开夹爪：点击 "Gripper Open"
4. 下降：向下推左侧摇杆
5. 关闭夹爪：点击 "Gripper Close"
6. 上升：向上推左侧摇杆
7. 移动到目标位置：使用左侧摇杆
8. 打开夹爪：点击 "Gripper Open"

### 使用吸泵抓取

1. 锁定机械臂：点击 "Lock Arm"
2. 移动到物体上方：使用左侧摇杆
3. 下降到物体：向下推左侧摇杆
4. 打开吸泵：点击 "Pump On"
5. 上升：向上推左侧摇杆
6. 移动到目标位置：使用左侧摇杆
7. 关闭吸泵：点击 "Pump Off"

## 摇杆控制说明

### 坐标系统
- **X轴**：左右方向（-1.0 到 1.0）
  - -1.0 = 最左
  - 0.0 = 中心
  - 1.0 = 最右

- **Y轴**：上下方向（-1.0 到 1.0）
  - -1.0 = 最上
  - 0.0 = 中心
  - 1.0 = 最下

### 速度控制
- 摇杆推得越远，速度越快
- 摇杆在中心时，速度为0
- 最大速度在摇杆边缘

### 视觉反馈
- **蓝色圆球**：正常状态
- **深蓝色圆球**：按下状态
- **实时位置显示**：显示当前摇杆位置

## 按钮颜色说明

- 🟢 **绿色**：夹爪和泵控制按钮
- 🔴 **红色**：机械臂锁定/释放按钮
- 🟠 **橙色**：重置按钮
- ⚪ **白色**：返回主菜单按钮

## 技术特点

### 虚拟摇杆
- 基于PyQt5自定义Widget
- 实时鼠标跟踪
- 平滑的视觉反馈
- 自动回中功能

### 控制精度
- 位置精度：0.01
- 更新频率：实时
- 响应延迟：< 50ms

### 安全特性
- 松开鼠标自动停止
- 重置功能快速停止
- 速度限制保护

## 注意事项

### 使用前检查
1. 确保机械臂已正确连接
2. 确保AGV小车已启动
3. 确保ROS环境正常运行
4. 检查硬件安全范围

### 安全提示
⚠️ **重要安全提示**：
- 首次使用前，先测试小幅度移动
- 注意机械臂的工作范围
- 避免碰撞障碍物
- 紧急情况下点击 "Release Arm" 释放机械臂
- 使用 "Reset" 按钮快速停止所有运动

### 常见问题

**Q: 摇杆不响应？**
A: 检查鼠标是否在摇杆区域内，确保点击了蓝色圆球

**Q: 机械臂不移动？**
A: 
1. 检查是否点击了 "Lock Arm"
2. 检查机械臂连接
3. 查看日志输出

**Q: AGV不移动？**
A:
1. 检查ROS是否运行
2. 检查导航是否启动
3. 查看日志输出

**Q: 如何紧急停止？**
A: 
1. 松开鼠标（摇杆自动回中）
2. 点击 "Reset to Start Position"
3. 点击 "Release Arm" 释放机械臂

## 与物理手柄的对比

| 特性 | 物理手柄 | 虚拟手柄 |
|------|---------|---------|
| 硬件需求 | 需要USB手柄 | 只需鼠标 |
| 精度控制 | 模拟摇杆 | 鼠标精确控制 |
| 视觉反馈 | 无 | 实时显示 |
| 便携性 | 需携带设备 | 软件内置 |
| 学习曲线 | 需熟悉按键 | 直观易用 |

## 开发说明

- 文件：`new_ui/virtual_joystick.py`

- 基于：PyQt5 QWidget
- 绘制：QPainter

### 信号
```python
positionChanged = pyqtSignal(float, float)
# 参数：(x, y) 范围 [-1.0, 1.0]
```

### 方法
```python
setPosition(x, y)  # 设置位置
reset()            # 重置到中心
```
