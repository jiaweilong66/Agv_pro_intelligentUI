# 新版UI集成说明文档

## 概述

新版UI (`new_ui/main_app.py`) 已经完全集成了原 `main.py` 的所有接口和功能，提供了更现代化的界面设计和更好的用户体验。

## 启动方式

### Windows
```bash
run_new_ui.bat
```

### Linux/Mac
```bash
python new_ui/main_app.py
```

## 使用流程

### 快速启动完整任务
1. 点击顶部菜单的 "Quick Start"
2. 点击左侧 "Quick Start" 按钮
3. 系统自动执行：
   - 启动雷达（如未启动）
   - 启动导航（如未启动）
   - 执行导航到货架
   - 执行停车充电
   - 显示任务完成

### 单独测试功能
1. 先启动雷达：点击 "Lidar - Open/Close"
2. 再启动导航：点击 "Navigation - Open/Close"
3. 测试其他功能：
   - 视觉识别：点击 ARUCO 或 QR 按钮
   - 运动控制：点击手柄或键盘控制
   - 任务执行：点击 Move to Shelf 或 Circular Sorting

## 状态指示

### 按钮颜色
- **蓝色**：默认状态，功能未激活
- **红色**：功能正在运行
- **绿色**：执行类按钮（Move to Shelf, Circular Sorting等）
- **白色**：返回按钮

### 任务节点
- **蓝色圆圈 + ...**：等待执行
- **绿色圆圈 + ✓**：已完成
- **红色圆圈 + ✗**：执行失败

### 连接线
- **绿色**：已完成的节点之间
- **灰色**：未完成的节点之间

## 错误处理

### 雷达未启动
- 尝试启动导航时会提示
- 尝试启动控制时会提示
- 需要先启动雷达

### 导航未启动
- 尝试执行任务时会提示
- 需要先启动导航

### 相机启动失败
- 日志会显示错误信息
- 检查相机连接
- 检查设备权限

## 调试模式

修改 `cofniger.py` 中的 `debug` 参数：
```python
debug = True  # 启用调试模式
```

调试模式会输出更详细的日志信息。
